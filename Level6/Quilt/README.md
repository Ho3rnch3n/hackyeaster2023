# Solution

well a picture with lots of qr codes...

luckily i already had to write a similar (but harder) script for hackvent 2022, so lets reuse this!

after running the script the first thing i noticed that every qr code only had one letter in it... (and luckily i scanned them in the right direction as it seems!)

so i concatted all the outputs together
